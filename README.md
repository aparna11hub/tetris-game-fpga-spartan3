# FPGA Tetris Game (Spartan-3)

## 📌 Project Description
This project implements the classic Tetris game on FPGA using Verilog HDL. The system uses VGA for display output and PS/2 keyboard for user input.

## ⚙️ Features
- Real-time VGA display
- Keyboard-based control
- Block movement and rotation
- Score display
- Pause and restart functionality

## 🎮 Controls
- 0 → Move Left  
- 1 → Move Right  
- 2 → Rotate Block  
- 3 → Move Down  
- R → Restart  
- P → Pause  

## 🖥️ Simulation
The design was simulated using VGA Playground, where VGA signals are directly visualized.

## 📂 Project Structure
- `src/` → Verilog source files  
- `simulation/` → Simulation details  
- `images/` → Output screenshots  
- `report/` → Project report
